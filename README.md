# 💻 Nikhil Raj — Interactive OS Portfolio

[![Hero Section Screenshot](./public/hero.png)](https://nikhilraj.dev)

Welcome to the source code of my interactive, operating system-inspired personal portfolio. Designed to be more than just a static resume, this application mimics a desktop environment within your browser—complete with draggable windows, a functioning terminal, 3D interactive bento grids, and an immersive cinematic hero section.

## ✨ Features

- **macOS-like Window Management:** Fully interactive desktop environment with draggable, resizable, and minimizable application windows.
- **Cinematic 3D Hero Section:** A highly immersive introduction featuring integrated Spline 3D objects and butter-smooth GSAP animations.
- **MagicBento™ Project Grid:** Modern and interactive project showcase using 3D hover effects.
- **Functional Terminal System:** A built-in, command-line interface allowing users to navigate the portfolio, read logs, and execute pre-defined commands.
- **Context-Aware Custom Cursor:** Intelligent cursor that morphs and provides precise visual feedback depending on the hovered element mode.
- **Seamless Communication:** Direct email integration straight from the desktop’s contact application using Resend/EmailJS.
- **Code-Rain & LaserFlow Aesthetics:** Dynamic background textures and animated section dividers to keep the interface feeling alive.

## 🛠️ Tech Stack

Built with cutting-edge tools to ensure blazing-fast performance, type-safety, and incredible developer experience:

- **Framework:** Next.js (App Router), React 19
- **Typing:** TypeScript
- **Styling:** Tailwind CSS v4
- **State Management:** Zustand
- **Animations:** Framer Motion, GSAP
- **3D & Graphics:** Three.js, React Three Fiber, React Drei, Spline
- **Email Providers:** Resend & EmailJS
- **Analytics:** Vercel Analytics & Speed Insights

## 🚀 How to Run Locally

Want to explore the code or try it out on your local machine?

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nikhiilraj/nikhil-raj-os-portfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd nikhil-raj-os-portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   # or 
   pnpm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to view the application.
